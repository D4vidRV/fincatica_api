# - Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```
3. Ejecutar el proyecto en modo desarrollo

```
yarn run start:dev
```
4. Levantar Base de Datos

```
docker-compose up -d
```
