# - Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Levantar Base de Datos

```
docker-compose up -d
```

4. Renombrar el archivo **.env.template** a **.env** y establecer variables de entorno

5. Ejecutar el proyecto en modo desarrollo

```
yarn run start:dev
```
